Sobre o "Simple 2048"

O "Simple 2048" é um jogo desenvolvido em HTML, CSS e Javascript. Seu objetivo principal é a criação de um aplicativo interessante para o usuário e a utilização deste sistema para o aprendizado da programação (por desenvolvedores que resolvam ver este código).

Como esse código ainda está em desenvolvimento, conforme forem implementadas novas funções, este arquivo será atualizado com caminhos e instruções acerca do trabalho executado! :D
